# Architecture Diagrams

This directory contains visual diagrams for AI system architectures.

## File Naming Convention

Use descriptive names that match the system case study:
- `[system-name]-architecture.png`
- `[system-name]-data-flow.png`
- `[system-name]-components.png`

## Diagram Guidelines

- Use clear, readable fonts
- Maintain consistent color scheme
- Include legends for complex diagrams
- Export as PNG or SVG for web compatibility
- Keep file sizes reasonable (< 1MB recommended)

## Tools

Common tools for creating architecture diagrams:
- Draw.io / diagrams.net
- Excalidraw
- Mermaid (code-based diagrams)
- Lucidchart
- Figma
