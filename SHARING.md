## üöÄ Quick Setup (If Not Already Done)

### Step 1: Push to GitHub

If you haven't already, push your portfolio to GitHub:

```bash
# If this is a new repository
git init
git add .
git commit -m "Initial portfolio setup"
git branch -M main
git remote add origin https://github.com/jasonhckim/jasons_portfolio.git
git push -u origin main
```

### Step 2: Enable GitHub Pages

1. Go to your repository on GitHub: `https://github.com/jasonhckim/jasons_portfolio`
2. Click **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under **Source**, select:
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Click **Save**
6. Wait 2-5 minutes for the first deployment

### Step 3: Verify It's Live

- Visit: `https://jasonhckim.github.io/jasons_portfolio/`
- You should see your portfolio homepage
- Check the **Actions** tab in your repository to see deployment status

## üÜò Troubleshooting

### Site Not Loading?
- Check **Actions** tab for build errors
- Verify repository is public (required for free GitHub Pages)
- Wait a few minutes for first deployment

### Changes Not Showing?
- Hard refresh browser (Cmd+Shift+R / Ctrl+Shift+R)
- Check GitHub Actions for deployment status
- Verify files were committed and pushed

### Need Custom Domain?
- In repository Settings ‚Üí Pages ‚Üí Custom domain
- Add your domain (e.g., `jasonkim.dev`)
- Update DNS records as instructed
