# Assets

This directory contains screenshots, sample outputs, and other visual assets for the portfolio.

## File Organization

Organize assets by system:
- `[system-name]/screenshot-1.png`
- `[system-name]/output-example.json`
- `[system-name]/demo-video.mp4`

## File Guidelines

- **Screenshots:** PNG format, reasonable resolution (1920x1080 max)
- **Outputs:** Include sample JSON, CSV, or text files that demonstrate system results
- **Videos:** Keep file sizes reasonable or host externally (YouTube, Loom)
- **Diagrams:** Prefer vector formats (SVG) when possible

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `document-classifier-output-sample.json`
- `email-routing-dashboard.png`
- `data-extraction-example.csv`
